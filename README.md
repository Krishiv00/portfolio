# Krishiv Goel - Portfolio

Personal portfolio site built from scratch with vanilla HTML, CSS, and JavaScript. No framework, no build step, no dependencies.

**Live site → [krishiv00.github.io/portfolio](https://krishiv00.github.io/portfolio/)**

---

## Stack

| Layer | Choice |
|-------|--------|
| Markup | Semantic HTML5 |
| Styling | Vanilla CSS with custom properties |
| Scripting | Vanilla JavaScript (ES6+) |
| Fonts | JetBrains Mono + Fraunces (Google Fonts) |
| Syntax highlighting | highlight.js |
| Hosting | GitHub Pages |

Zero npm. Zero build step. Zero frameworks.

---

## Features

- Terminal-style hero with a typewriter animation cycling through programmer phrases
- Scroll-driven reveal animations via `IntersectionObserver`
- Staggered project card entrance animations
- Active nav link tracking on scroll with suppression during click navigation
- Fully responsive layout with a hamburger mobile menu
- Syntax-highlighted C++ and Bash code blocks
- PWA manifest + full favicon set for all platforms
- SEO-optimised: meta description, Open Graph, Twitter Card, JSON-LD structured data, canonical URL, `robots.txt`, `sitemap.xml`
